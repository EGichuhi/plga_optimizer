
# PLGA Drug Delivery Optimizer

The PLGA Drug Delivery Optimizer is a tool for simulating and optimizing drug release from PLGA (poly(lactic‑co‑glycolic acid)) polymer systems.  
It is designed so that it can use a simple command‑line interface (CLI) or an optional Streamlit app.

This project runs automatically in **GitHub Codespaces**, meaning you don’t need to install Python, Conda, or RDKit on your own computer.  
Just open the Codespace and run the optimizer.

---

## 🌟 What This Tool Does

PLGA is a biodegradable polymer widely used in controlled drug delivery.  
The optimizer helps you:

- Predict drug release profiles based on PLGA composition  
- Estimate degradation and diffusion behavior   
- Optimize formulation parameters  
- Export results for analysis  

Everything is wrapped in a simple CLI so you don’t need to write code.

---

## 🚀 Quick Start (No Installation Needed)

### **1. Open the project in GitHub Codespaces**

On the GitHub repo page:

1. Click the green **“Code”** button  
2. Choose **“Open with Codespaces”**  
3. Select **“New Codespace”**

The environment will automatically:

- Install Conda  
- Create the `plga_venv` environment  
- Install RDKit  
- Install all Python dependencies  
- Activate the environment for you  

You don’t need to do anything else.

---

## ▶️ Running the Optimizer (CLI)

Once the Codespace finishes setting up, open the terminal and run:

**python plga_optimizer.py**
